# React Issues
A simple exercise which gets some React issues from its official repository and display them in a styled table.

### Run application
1. `git clone https://github.com/ecmeirelles/ReactIssues.git`
2. `cd ReactIssues`
3. `yarn install`
4. `yarn start`
